DESCRIPTION
* This Java application implements the stack data structure by reading IDs and names from a text file (emp.txt) and using the scanner function to manipulate the data. 
The hasNextLine() method checks if another line of input is needed, and the line.split("\\s+"); method splits the line into an array of strings. 
The Class EmployeeStack methods will manipulate the stack with the data from the text file.

FROM TERMINAL
* Type on your terminal the following command line: cd [location of the file]

Example:
cd /Users/[your user name]/Downloads

* The jar application should run without any problem.


NOTES
* * * The text file must be on the same location as the .jar file
